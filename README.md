# merkle

`merkle` is a small Go library that implements several cryptographic constructs.

Written primarily for educational purposes rather than production `:-)`.

## Install

```bash
go get github.com/butcher-of-blaviken/merkle
```

## Testing

```bash
go test -cover
```
